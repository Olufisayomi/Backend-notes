# Variables in Python

Variables are used to store data. They are like a container for values. Variables make code easier to read, reuse and update. Variables that can be changed are called mutable variables while the ones that can't be changed are called iimutable variables. 

Example:

```python
name = "Kehinde"
age = 18
```
In the example above, name stores text and age stores a number.

## Rules for naming variables
- Variable names cannot start with numbers.
- Variable names should not contain spaces.
- Use only meaningful names.

## Values variables can store
- Strings
- integers
- Floats
- Boolean
